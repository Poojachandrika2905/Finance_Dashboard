# 💰 Finance Dashboard UI

## 📌 Overview
This is a responsive Finance Dashboard built as part of a frontend assignment. 
It helps users track income, expenses, and financial insights in a clean and interactive interface.

---

## 🚀 Features

### 📊 Dashboard
- Total Balance, Income, Expenses cards
- Balance trend (Line Chart)
- Spending breakdown (Pie Chart)

### 📋 Transactions
- View all transactions
- Search by category/date/type
- Filter (Income / Expense)
- CSV export

### 👤 Role-Based UI
- Viewer → Read-only access
- Admin → Add/Edit buttons (UI simulation)

### 📈 Insights
- Highest spending category
- Monthly comparison
- Last month expenses
- Financial observation

---

## 🎨 UI/UX
- Fully responsive (mobile, tablet, desktop)
- Dark / Light mode toggle
- Clean and modern layout
- Smooth transitions

---

## ⚙️ Tech Stack
- React
- Tailwind CSS
- Recharts

## 💾 Additional Features
